# Wedding Invite Demo (Product Launch Style)

Open `index.html` in any browser. Customize the details by editing the `WEDDING` block in the Developer Notes or by updating the text in the HTML.

## Features
- Launch-banner design with dark/light theme toggle (bottom-right)
- Live countdown to Feb 25, 2026 @ 10:40 PM CST
- RSVP demo saves entries to `localStorage` (no server)
- Easter egg in the browser console: type `party()` for confetti
- Fully offline, one HTML file

## Customize
- Date/time: update the ISO string in the countdown target (`2026-02-26T04:40:00Z`).
- Sections: edit headings & paragraphs under Release Notes / Schedule / RSVP.
- Colors: tweak CSS variables in `:root`.
